# ⚡ Pokedex App

Ein interaktiver Pokedex, der Daten von der offiziellen [PokeAPI](https://pokeapi.co/) abruft. Dieses Projekt wurde im Rahmen meiner Lernphase zum Frontend-Entwickler erstellt.



## 🚀 Live Demo
Schau dir die fertige App hier an: 
[https://blitzbombong.github.io/myPokedex/](https://blitzbombong.github.io/myPokedex/)

## ✨ Features
- **Dynamisches Laden:** Daten werden direkt von der REST-API abgerufen.
- **Suchfunktion:** Filtere Pokémon nach Namen in Echtzeit.
- **Detail-Ansicht:** Klicke auf eine Karte, um Statistiken (HP, Attack, Defense) in einem Overlay anzuzeigen.
- **Responsive Design:** Optimiert für Desktop und mobile Endgeräte.
- **Infinite Scrolling:** Lädt automatisch weitere Pokémon beim Herunterscrollen.

## 🛠️ Verwendete Technologien
- **HTML5 & CSS3:** Struktur und modernes Styling (Flexbox/Grid).
- **JavaScript (Vanilla):** DOM-Manipulation, Asynchrones Programmieren (Fetch API, Async/Await).
- **PokeAPI:** Externe Datenquelle für alle Pokémon-Informationen.

## 📖 Was ich gelernt habe
In diesem Projekt konnte ich mein Wissen in folgenden Bereichen vertiefen:
- Umgang mit **Asynchronem JavaScript** und dem Verarbeiten von JSON-Daten.
- Erstellung von **modalen Dialogen** ohne externe Libraries.
- Strukturierung von CSS in verschiedene Dateien (`card.css`, `dialog.css`, etc.) für bessere Wartbarkeit.
